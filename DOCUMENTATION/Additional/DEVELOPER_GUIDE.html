دليلك الشامل للبدء في تطوير نظام قيمّ التعليمي

🚀 البدء السريع

1. متطلبات النظام

# Required PHP >= 8.2 Composer >= 2.5 Node.js >= 18.x NPM >= 9.x MySQL >= 8.0 Redis >= 7.0 # Recommended Git >= 2.40 VS Code with extensions

2. إعداد البيئة

# Clone repository git clone https://github.com/qiyamm/qiyamm-platform.git cd qiyamm-platform # Install PHP dependencies composer install # Install Node dependencies npm install # Copy environment file cp .env.example .env # Generate app key php artisan key:generate # Run migrations php artisan migrate --seed # Build assets npm run dev

3. تشغيل التطبيق

# Terminal 1: Laravel Server php artisan serve # Terminal 2: Vite Dev Server npm run dev # Terminal 3: Queue Worker php artisan queue:work # Access: http://localhost:8000
