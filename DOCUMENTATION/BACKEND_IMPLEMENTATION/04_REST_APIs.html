🔐Authentication APIs

POST /auth/login

تسجيل الدخول والحصول على JWT Token

Request Body
{
  "email": "user@example.com",
  "password": "password123"
}
Response 200 OK
{
  "message": "Login successful",
  "data": {
    "user": { "id": 1, "name": "أحمد", "role": "student" },
    "token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
    "expires_in": 3600
  }
}
POST /auth/register

تسجيل مستخدم جديد

{
  "name": "محمد علي",
  "email": "mohamed@example.com",
  "password": "SecurePass123!",
  "password_confirmation": "SecurePass123!",
  "role": "student",
  "token_code": "SCH-2024-XYZ"
}
POST /auth/logout

تسجيل الخروج وإلغاء الـ Token

👥Users APIs

GET /users

عرض جميع المستخدمين مع Pagination و Filters

Query Parameters
?page=1&per_page=15&role=student&school_id=5&status=active&search=أحمد
Response 200 OK
{
  "data": [
    {
      "id": 1,
      "name": "أحمد محمد",
      "email": "ahmed@example.com",
      "role": "student",
      "school": { "id": 5, "name": "مدرسة النور" },
      "points": 1250,
      "coins": 500,
      "status": "active"
    }
  ],
  "meta": {
    "current_page": 1,
    "total": 150,
    "per_page": 15,
    "last_page": 10
  }
}
GET /users/{id}

عرض مستخدم محدد بالتفاصيل

POST /users

إنشاء مستخدم جديد (Super Admin / School Admin)

PUT /users/{id}

DELETE /users/{id}
