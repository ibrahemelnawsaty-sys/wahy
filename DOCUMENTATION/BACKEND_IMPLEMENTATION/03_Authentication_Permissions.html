🛡️Middleware Implementation

CheckRole Middleware

<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class CheckRole
{
    public function handle(Request $request, Closure $next, ...$roles)
    {
        if (!$request->user()) {
            return response()->json(['message' => 'Unauthenticated'], 401);
        }

        $userRole = $request->user()->role->name;

        if (!in_array($userRole, $roles)) {
            return response()->json([
                'message' => 'Unauthorized. Required role: ' . implode(', ', $roles)
            ], 403);
        }

        return $next($request);
    }
}

// Usage in routes/api.php
Route::middleware(['auth:sanctum', 'role:super_admin'])->group(function () {
    Route::resource('schools', SchoolController::class);
    Route::resource('tokens', TokenController::class);
});

Route::middleware(['auth:sanctum', 'role:teacher,school_admin'])->group(function () {
    Route::get('submissions/pending', [SubmissionController::class, 'pending']);
    Route::post('submissions/{id}/correct', [SubmissionController::class, 'correct']);
});

CheckPermission Middleware

<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class CheckPermission
{
    public function handle(Request $request, Closure $next, string $permission)
    {
        $user = $request->user();

        if (!$user || !$user->role->hasPermission($permission)) {
            return response()->json([
                'message' => 'Forbidden. Required permission: ' . $permission
            ], 403);
        }

        return $next($request);
    }
}

// Usage
Route::middleware(['auth:sanctum', 'permission:manage_content'])->group(function () {
    Route::resource('values', ValueController::class);
    Route::resource('concepts', ConceptController::class);
});

🔒Security Features

1. Rate Limiting

تحديد عدد المحاولات لتجنب Brute Force Attacks

  • ✓ Login: 5 محاولات كل 15 دقيقة
  • ✓ Register: 3 محاولات كل ساعة
  • ✓ APIs: 60 طلب في الدقيقة

2. CSRF Protection

حماية ضد Cross-Site Request Forgery

  • ✓ CSRF Token في جميع POST/PUT/DELETE requests
  • ✓ Laravel Sanctum CSRF Cookie

3. Password Hashing

تشفير كلمات المرور باستخدام Bcrypt

  • ✓ Bcrypt with cost factor 12
  • ✓ Never store plain passwords

4. Email Verification

التحقق من البريد الإلكتروني قبل التفعيل

  • ✓ Send verification email on register
  • ✓ Restrict access until verified

5. Two-Factor Authentication (2FA)

مصادقة ثنائية اختيارية للأمان الإضافي

  • ✓ OTP via SMS
  • ✓ TOTP (Google Authenticator)
  • ✓ Backup codes
